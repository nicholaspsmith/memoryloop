### Mobile (< 768px)

- Stack goal cards vertically
- Skill tree scrolls horizontally
- Study cards take full width
- Bottom navigation bar

### Tablet (768px - 1024px)

- 2-column goal grid
- Skill tree fits in viewport
- Side-by-side metrics

### Desktop (> 1024px)

- 3-column goal grid
- Full skill tree visualization
- Sidebar navigation
